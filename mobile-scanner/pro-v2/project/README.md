# Autoguard Mobile Scanner Pro v2.0

Aplicación Android profesional para diagnóstico OBD-II móvil.

## Incluye
- WiFi ELM327 prioritario (`192.168.0.10:35000`).
- Bluetooth Classic con dispositivos enlazados.
- Simulador para pruebas sin vehículo.
- Lectura de DTC confirmados, pendientes y permanentes.
- Borrado Mode 04 con confirmación.
- Datos en vivo con selección de sensores.
- PIDs ampliados: RPM, velocidad, refrigerante, IAT, carga, mariposa, MAF, MAP, O2, STFT, LTFT, avance, presión y nivel de combustible, barométrica, runtime, voltaje ECU, ambiente y purga EVAP.
- Pruebas funcionales guiadas para ventilador, ventanas, cierre centralizado, luces y sensores.
- Ficha opcional del vehículo.
- Informe PDF guardado en Descargas/Autoguard.
- Demostración Pro de 7 días.
- Verificación de licencias firmadas RSA preparada para el panel administrativo Autoguard.

## Limitación técnica
Un ELM327 genérico no ofrece de forma universal pruebas activas bidireccionales de módulos de carrocería. Las activaciones reales requieren hardware J2534/UDS o interfaz OEM y perfiles específicos de cada marca.
