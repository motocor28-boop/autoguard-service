# Autoguard Mobile Scanner Pro v2.0

Versión profesional Android en desarrollo.

## Alcance
- WiFi ELM327 prioritario, Bluetooth Classic y simulador.
- Lectura y borrado confirmado de DTC.
- Datos en vivo con selección de sensores y PIDs ampliados.
- Pruebas funcionales guiadas para ventilador, ventanas, cierre centralizado y luces.
- Ficha opcional de cliente y vehículo.
- Informes PDF profesionales compartibles.
- Historial local de diagnósticos.
- Arquitectura preparada para licencias administradas por Autoguard.

Las pruebas activas bidireccionales reales dependerán de hardware y perfiles OEM compatibles; un ELM327 genérico se mantiene limitado a diagnóstico OBD-II y procedimientos guiados.
