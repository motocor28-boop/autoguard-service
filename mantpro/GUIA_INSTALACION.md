# MANTPRO IA Cloud — instalación y uso

Dirección oficial: https://motocor28-boop.github.io/autoguard-service/mantpro/

## Computador Windows

1. Abra la dirección oficial en Microsoft Edge.
2. Ingrese `motocor28@gmail.com` y abra el enlace recibido desde Supabase.
3. En Edge seleccione **… > Aplicaciones > Instalar este sitio como una aplicación**.
4. Active el acceso directo del escritorio y, si lo desea, ancle MANTPRO a la barra de tareas.

No se instala un servidor, base de datos, PowerShell ni archivos CMD. La información queda en Supabase Cloud y se conserva temporalmente en el equipo cuando no hay internet.

## Teléfono Android

1. Abra la misma dirección oficial en Google Chrome.
2. Ingrese `motocor28@gmail.com` y abra el enlace recibido desde Supabase en el mismo teléfono.
3. En Chrome seleccione **⋮ > Agregar a pantalla principal > Instalar**.
4. Autorice cámara y fotografías cuando MANTPRO lo solicite.

## Registro desde ambos equipos

El computador y el teléfono muestran los mismos datos. En ambos se pueden crear OT, registrar tiempos reales, avances, desviaciones, caminatas, técnicos, KPI y fotografías. Al recuperar internet, los registros pendientes se sincronizan automáticamente.

## Fotografías e informes

En cada avance o desviación use **Tomar o adjuntar fotografía** y seleccione el tipo de evidencia. El informe por OT reúne automáticamente las fotografías iniciales, de avance, hallazgo, desviación y cierre, con su fecha y descripción.

## Respaldo

En **Más > Elegir carpeta de respaldos**, seleccione una carpeta externa. MANTPRO crea la estructura `Respaldos_MANTPRO_IA/Año/Mes` en navegadores compatibles. Si el navegador no permite elegir carpetas, descarga el archivo JSON para guardarlo manualmente.
